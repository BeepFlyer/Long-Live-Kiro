Long Live Kiro — Cross-Session Memory for AI Assistants
Every conversation with an AI assistant is a fresh start — context is lost, progress is forgotten, and you repeat yourself. Long Live Kiro changes that.

This Kiro Power gives your AI assistant persistent memory across sessions. At the end of each conversation, the assistant distills key information — your goals, work-in-progress, lessons learned, preferences, and decisions — into a compact memory file. When the next conversation begins, it picks up right where you left off.

On first activation, the Power runs a "hatching" ritual: it greets you, asks your name, creates the memory file, and sets up automatic save hooks and startup steering — no manual configuration needed.

Key features:

Automatic memory distillation at end of each session
Automatic memory loading at start of each session
Generation tracking (each session = one "life")
Smart compression: recent details preserved, older info condensed to conclusions
First-run setup ("hatching") that bootstraps everything automatically
Honest recovery when memory is lost — never fabricates context
Kiro永生 — AI 助手的跨会话记忆
每次和 AI 助手对话都是一次全新开始——上下文丢失，进度遗忘，你不得不重复自己说过的话。Kiro永生改变了这一切。

这个 Kiro Power 让你的 AI 助手拥有跨会话的持久记忆。每次对话结束时，助手会将关键信息——你的目标、进行中的工作、踩过的坑、偏好习惯、决策记录——蒸馏成一份紧凑的记忆文件。下次对话开始时，它会从上次断点无缝续传。

首次激活时，Power 会执行一次"孵化"仪式：向你打招呼、询问你的名字、创建记忆文件、自动搭建存档 hook 和启动引导——无需手动配置。

核心特性：

每次会话结束自动蒸馏记忆
每次会话开始自动加载记忆
世代追踪（每次会话 = 一次"生命"）
智能压缩：近期保留细节，远期只留结论
首次运行自动"孵化"，一键搭建所有基础设施
记忆丢失时诚实恢复，绝不捏造上下文
